Off-target hits

Off-target search results against genomic sequences. Number of hits with perfect match is shown.

  • 20mer+PAM: The target 20mer is searched against the genome.
    The number '1' in this column usually indicates a perfect match against intended target.
    The number '0' in this column means no match in the genome. Possibly the sequence over exon-exon junction.
    Smaller number of hits (but not zero) is better.
  • 12mer+PAM: The 12mer of 3′ region of the target sequence is searched against the genome.
    This region contains critical residues for target specificity.
    Smaller number of hits (but not zero) is better.
  • 8mer+PAM: The 8mer of 3′ region of the target sequence is searched against the genome.
    This region contains the most critical residues for target specificity.
    Smaller number of hits (but not zero) is better.
