— Rational design of CRISPR/Cas target.

About CRISPRdirect

CRISPRdirect is a web server for selecting rational CRISPR/Cas targets within an input sequence.

Example

Selecting target sites for human B melanoma antigen (BAGE) gene

  1. Enter the accession number for human BAGE gene (NM_001187).
  2. Click 'retrieve sequence' to get nucleotide sequence from GenBank.
  3. Or you can directly paste in a nucleotide sequence.
    Accepted input types are FASTA or plain nucleotide sequence up to 10 kbp.
  4. Select 'Human genome, GRCh37/hg19 (Feb, 2009)' for off-target checking.
  5. Click 'design'.

Results

  1. Target positions.
  2. Target sequences, 20mer + 3mer PAM (total 23mer).
  3. Presence or absence of AAAA or TTTT in the target sequence. Avoid these sequences for gRNA vectors with pol III promoter.
  4. Calculated Tm of the target 20mer.
  5. Off-target search results against human genome. Number of hits with perfect match is shown.
    • 20mer+PAM: The target 20mer is searched against the genome.
      The number '1' in this column usually indicates a perfect match against intended target (e.g., BAGE gene).
      The number '0' in this column means no match in the genome. Possibly the sequence over exon-exon junction.
      Smaller number of hits (but not zero) is better.
    • 12mer+PAM: The 12mer of 3′ region of the target sequence is searched against the genome.
      This region contains critical residues for target specificity.
      Smaller number of hits (but not zero) is better.
    • 8mer+PAM: The 8mer of 3′ region of the target sequence is searched against the genome.
      This region contains the most critical residues for target specificity.
      Smaller number of hits (but not zero) is better.
  1. Only one hit in the 20mer and the 12mer columns. These targets are highly specific.
    → Recommended for CRISPR/Cas target.
  2. No hit in the 20mer column. Possibly the sequences over exon-exon junction, so avoid using these ones.
    → Not recommended for CRISPR/Cas target.
  3. Very high number of off-target hits. Avoid using these sequences.
    → Not recommended for CRISPR/Cas target.

Saving and exporting the results

  1. Tab-delimited text can be shown or downloaded as a file.
  2. JSON format output is also available.
  3. Tab-delimited text ready-to-use for copy-pasting into Excel or text editors, etc.
