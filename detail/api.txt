GGGenome REST API

Query format:

http://GGGenome.dbcls.jp/DB/K/SEQUENCE[.FORMAT][.download]
  DB (optional):
    'hg19' - Human (Homo sapiens) genome, GRCh37/hg19 (Feb, 2009) or
    'mm10' - Mouse (Mus musculus) genome, GRCm38/mm10 (Dec, 2011) or
    'rn5'  - Rat (Rattus norvegicus) genome, RGSC 5.0/rn5 (Mar, 2012) or
    'galGal4' - Chicken (Gallus gallus) genome, ICGSC Gallus_gallus-4.0/galGal4 (Nov, 2011) or
    'xenTro3' - Frog (Xenopus tropicalis) genome, JGI 4.2/xenTro3 (Nov, 2009) or
    'danRer7' - Zebrafish (Danio rerio) genome, Zv9/danRer7 (Jul, 2010) or
    'ci2'  - Sea squirt (Ciona intestinalis) genome, JGI 2.1/ci2 (Mar, 2005) or    
    'dm3'  - Fruit fly (Drosophila melanogaster) genome, BDGP R5/dm3 (Apr, 2006) or
    'ce10' - Roundworm (Caenorhabditis elegans) genome, WS220/ce10 (Oct, 2010) or
    'TAIR10' - Thale cress (Arabidopsis thaliana) genome, TAIR10 (Nov, 2010) or
    'rice' - Rice (Oryza sativa) genome, Os-Nipponbare-Reference-IRGSP-1.0 (Oct, 2011) or
    'bmor1' - Silkworm (Bombyx mori) genome, Bmor1 (Apr, 2008) or
    'sacCer3' - Budding yeast (Saccharomyces cerevisiae) (S288C) genome, sacCer3 (Apr, 2011) or
    'refseq' - RefSeq complete RNA, release 65 (May, 2014) or
    'prok' - Prokaryotic TogoGenome from RefSeq 62 (Nov, 2013) or
    'ddbj' - DDBJ release 92.0 (Feb, 2013)
    Default: hg19
  K (optional):
    Maximum number of mismatches/gaps.
    Default: 0
  SEQUENCE:
    Nucleotide sequence, case insensitive.
  FORMAT (optional):
    'html' - HTML
    'txt'  - Tab-delimited text
    'json' - JSON object
    Default: html
  download (optional):
    Download result as a file.

Examples:

http://GGGenome.dbcls.jp/TTCATTGACAACATT
  Search 'TTCATTGACAACATT' in human genome hg19. (default)
  Search only perfect match sequences. (default)
  Output in HTML format. (default)

http://GGGenome.dbcls.jp/mm10/2/TTCATTGACAACATTGCGT.txt
  Search 'TTCATTGACAACATTGCGT' in mouse genome mm10.
  Search sequences within two mismatches and gaps.
  Output in tab-delimited text format.
